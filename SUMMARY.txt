                         QUICK START
================================================================================

Choose ONE of these:

GROQ (Easiest):
  1. Go to https://console.groq.com
  2. Sign up (FREE, no credit card)
  3. Get API key
  4. Set: export GROQ_API_KEY='your-key'
  5. Run: streamlit run app_streamlit_free.py
  6. Done!

OLLAMA (Local & Private):
  1. Download from https://ollama.ai
  2. Run: ollama serve
  3. Run: ollama pull mistral
  4. Run: streamlit run app_streamlit_free.py
  5. Done! 

HUGGINGFACE (Open Source):
  1. Install: pip install transformers torch
  2. Run: streamlit run app_streamlit_free.py
  3. Wait for model download (first time)
  4. Done!
